## What's in this bundle?

This folder contains everything you need to create presentations in Google NotebookLM:

1. **PACK_RENDER.html** - Complete pitch pack as a single HTML document
2. **PACK_RENDER.pdf** - PDF version (if available)
3. **PRESENTATION_BRIEF.md** - Talk tracks and presentation notes
4. **ASSET_MANIFEST.yml** - Visual asset specifications (if included)
5. **DIALOGUE.yml** - Character voice and dialogue specs (if included)
6. **NOTEBOOKLM_STUDIO_PROMPTS.txt** - Ready-to-use prompts for NotebookLM

## Quick Start

1. Go to [NotebookLM](https://notebooklm.google.com/)
2. Create a new notebook
3. Upload **PACK_RENDER.html** (or PDF) as a source
4. Upload **PRESENTATION_BRIEF.md** as a source
5. Open **NOTEBOOKLM_STUDIO_PROMPTS.txt** and copy a prompt
6. Paste into NotebookLM and let it generate!

## Recommended Outputs

- **Slide Deck**: Great for stakeholder presentations
- **Audio Overview**: Perfect for async reviews (6-12 min)
- **Video Overview**: Ideal for team onboarding (2-4 min)
- **Mind Map**: Useful for brainstorming sessions

## Tips

- Start with the slide deck prompt for a quick overview
- Use the audio overview for detailed walkthroughs
- The mind map helps identify scope creep early
- Regenerate with different prompts to explore variations
